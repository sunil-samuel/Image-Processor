Huggingface Token

A Huggingface token is required to run the AI models that are responsible for generating the descriptive text for images.

Visit the following URL to generate the token: https://huggingface.co/settings/tokens

Once the token is generated, use File->Set Huggingface Token and place this newly created token into the text box. This will persist the token into your personal folder so that the next time you run this application, the token is already set.

Processing Options

The 'Processing Options' are configuration paramters as follows:

  • Add Created Date: Inspect the image for the 'date taken' information using the EXIF meta data. If this date exist, then the file is prefixed with this date. If the 'date taken' does not exist, then the OS file creation date is used.
  • Open Sub-Folders: Traverse the 'Source Directory' and any sub folders within this directory. Otherwise, just load images from the 'Source Directory'.
  • Move Files or Copy Files: NOTE: Only either 'Move Files' or 'Copy Files' can be selected. If selected, then the images are moved or copied to the destination directory selected within the 'Destination Directory Selection' area. Year directories with the format YYYY will be created to store the files within the corresponding years.
  • Create Month Folder: NOTE: Only enabled is either 'Move Files' or 'Copy Files' is selected. Create month will create another sub-folder within the year folder for the month in the format MM-MonthName, e.g. 01-January
  • AI Description: Use huggingface models to dynamically describe the the images. These descriptions are then stored within the EXIF metadata within the UserComment section.

    The level of detail of the description that is created can be selected.
    Brief: The caption is brief and short
    Standard: The caption is more detailed
    Full: The caption has lots of detail of the image

Source Directory Selection

Select the directory path to where all of the images are stored. The application will traverse this directory and analyze each image.

Destination Directory Selection

If you selected 'Move Files' or 'Copy Files', then provide the directory where these images will be moved or copied. The files will be moved to a directory that corresponds with the 'date taken' year, with the format YYYY. If 'Create Month Folder' is selected, then there will be another sub-folder with the month.
